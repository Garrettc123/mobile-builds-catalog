# Contributing Guidelines

Thank you for contributing to mobile-builds-catalog!

## Adding New Builds

1. Create version directory
2. Add build files
3. Include release notes
4. Update CHANGELOG.md
5. Submit PR

## Build Naming Convention

```
app-name-v1.0.0-release.apk
app-name-v1.0.0-debug.apk
```

## Version Control

- Follow semantic versioning
- Tag releases properly
- Include checksums
